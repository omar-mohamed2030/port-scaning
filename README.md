# Simple Python Port Scanner

A lightweight and efficient TCP port scanner built with Python's `socket` and `sys` modules. This tool is designed for educational purposes and basic network reconnaissance.

## ⚙️ Features
- Scans a specific range of ports (50-80).
- Resolves hostnames to IP addresses.
- Displays the scan start time.
- Handles user interruptions (Ctrl+C) gracefully.
